# Fab libraries (JLCPCB stackups + design rules)

These folders help you pick a **correct PCB stackup** before routing (especially for RF / ESP / LoRa boards).

## What's here

| Path | Source | Use |
|------|--------|-----|
| `recommended/` | Curated subset | **Start here** — files the Altium MCP panel opens |
| `jlcpcb_autogenerated_stackups/` | [gsuberland/jlcpcb_autogenerated_stackups](https://github.com/gsuberland/jlcpcb_autogenerated_stackups) | Full JLCPCB Altium `.stackupx` set (auto-generated from JLC API) |
| `jlcpcb-design-rules-stackups/` | [ayberkozgur/jlcpcb-design-rules-stackups](https://github.com/ayberkozgur/jlcpcb-design-rules-stackups) | Hand-curated `.stackup` + `.RUL` for 2/4/6L JLCPCB |

The large vendor clones are gitignored; `recommended/` is what the plugin ships/deploys.

## Is the data correct?

- **Autogenerated stackups**: good for matching JLCPCB's published impedance templates, but the upstream author warns they are **not manually verified** and JLC data sometimes has typos. Always cross-check thickness / Dk against [JLCPCB impedance](https://jlcpcb.com/impedance) before controlled-impedance fab.
- **ayberkozgur rules/stackups**: smaller, human-curated for common JLC caps (e.g. 4L JLC7628 1.6 mm). Prefer these for **fab rule mins** (trace/via); prefer `.stackupx` for modern Altium Layer Stack Manager.
- For impedance-controlled builds, order **FR-4 TG155** (NP-155F), not the cheap TG135 default.

## What you should use (typical boards)

| Board type | Layers | Template | Notes |
|------------|--------|----------|-------|
| Simple MCU, no RF | 2L 1.6 mm 1 oz | `2layer-1oz-1.6mm.stackup` | Cheap; poor return path for RF/USB |
| **ESP + LoRa / WiFi (recommended)** | **4L 1.6 mm outer 1 oz / inner 0.5 oz** | `JLC04161H-7628.stackupx` | L2 = solid GND, L3 = 3V3 pour, Top/Bottom signals |
| Dense BGA / many supplies | 6L 1.6 mm | `JLC06161H-7628.stackupx` | Extra planes / routing |

### Correct 4-layer assignment for RF + digital

1. **Top** — components, RF microstrip, short logic  
2. **Mid1 (L2)** — **solid GND** (never split under RF)  
3. **Mid2 (L3)** — **3V3** (and other power) pours  
4. **Bottom** — GND pour + leftover routing  

Do **not** route +5 under a GND thermal pad on Bottom (that shorted U5 on Duel_Esp_LoRa).

## How to load in Altium

1. Open the PCB → **Design → Layer Stack Manager**  
2. **File → Load Stackup From File…**  
3. Pick a `.stackupx` from `recommended/` (or Documents\AltiumEE\recommended-stackups after using the panel)  
4. Confirm copper/dielectric thicknesses match the JLC order page  
5. Then run **Setup Net Classes & Rules** in the MCP panel  

Optional fab mins: Design → Rules → load the matching `.RUL` from `recommended/` (2/4/6 layer-1oz).
